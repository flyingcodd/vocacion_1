Politicas de privacidad

1. Las contraseñas

Las contraseñas son almacenadas encriptadas en la base de datos, por lo que no es posible recuperarlas. Si el usuario olvida su contraseña, deberá solicitar una nueva.

2. Los datos personales

Los datos personales son almacenados en la base de datos y son utilizados únicamente para fines de autenticación y autorización. La entidad pública del estado se compromete a no compartir los datos personales con terceros.

3. Los datos de navegación

Los datos de navegación son almacenados en la base de datos y son utilizados únicamente para fines estadísticos. La entidad pública del estado se compromete a no compartir los datos de navegación con terceros.
